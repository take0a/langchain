# LangChain パッケージのインストール方法

LangChain エコシステムはさまざまなパッケージに分かれており、インストールする機能を正確に選択できます。

## 正式リリース

メインの `langchain` パッケージをインストールするには、次のコマンドを実行します。

import TabItem from '@theme/TabItem';
import CodeBlock from "@theme/CodeBlock";

<Tabs>
  <TabItem value="pip" label="Pip" default>
    <CodeBlock language="bash">pip install langchain</CodeBlock>
  </TabItem>

  </TabItem>
</Tabs>

このパッケージはLangChainを使うための健全な出発点として機能しますが、LangChainの価値の多くは、様々なモデルプロバイダーやデータストアなどと統合することで発揮されます。
デフォルトでは、統合に必要な依存関係はインストールされていません。特定の統合に必要な依存関係は、以下に示すように個別にインストールする必要があります。

## エコシステムパッケージ

`langsmith` SDKを除き、LangChainエコシステム内のすべてのパッケージは`langchain-core`に依存しています。`langchain-core`には、他のパッケージが使用する基本クラスと抽象化が含まれています。
以下の依存関係グラフは、さまざまなパッケージの関連性を示しています。
矢印は、ソースパッケージがターゲットパッケージに依存していることを示します。

![](/img/ecosystem_packages.png)

パッケージをインストールする際、そのパッケージの明示的な依存関係（`langchain-core` など）を明示的にインストールする必要はありません。
ただし、その依存関係の特定のバージョンでのみ利用可能な機能を使用する場合は、明示的にインストールすることを選択できます。
その場合、インストールまたは固定されたバージョンが、使用する他の統合パッケージと互換性があることを確認する必要があります。

### LangChain core
`langchain-core`パッケージには、LangChainエコシステム全体で使用されている基本的な抽象化と、LangChain Expression Languageが含まれています。`langchain`によって自動的にインストールされますが、個別に使用することもできます。以下のコマンドでインストールします。

```bash
pip install langchain-core
```

### 統合パッケージ

OpenAIやAnthropicなどの特定の統合には、独自のパッケージがあります。
独自のパッケージを必要とする統合については、[統合ドキュメント](/docs/integrations/providers/)にその旨が記載されています。
すべての統合パッケージの一覧は、[APIリファレンス](https://python.langchain.com/api_reference/)の「パートナーライブラリ」ドロップダウンから確認できます。
これらのパッケージをインストールするには、次のコマンドを実行します。

```bash
pip install langchain-openai
```

独自のパッケージに分割されていない統合は、`langchain-community` パッケージに含まれます。以下のコマンドでインストールします。

```bash
pip install langchain-community
```

### LangChain 実験版
`langchain-experimental` パッケージには、研究および実験用途を目的とした実験的な LangChain コードが含まれています。
インストール方法:

```bash
pip install langchain-experimental
```

### LangGraph
`langgraph` は、LLM を用いたステートフルなマルチアクターアプリケーションを構築するためのライブラリです。LangChain とスムーズに統合されますが、LangChain なしでも使用できます。
インストール方法:

```bash
pip install langgraph
```

### LangServe
LangServe は、開発者が LangChain のランナブルとチェーンを REST API としてデプロイするのに役立ちます。
LangServe は LangChain CLI によって自動的にインストールされます。
LangChain CLI を使用しない場合は、次のコマンドでインストールします:

```bash
pip install "langserve[all]"
```
クライアントとサーバーの両方の依存関係に対して。または、クライアントコードの場合は `pip install "langserve[client]"` 、サーバーコードの場合は `pip install "langserve[server]"` を実行します。

### LangChain CLI
LangChain CLI は、LangChain テンプレートやその他の LangServe プロジェクトを操作するのに便利です。
以下を使用してインストールします:

```bash
pip install langchain-cli
```

### LangSmith SDK
LangSmith SDKはLangChainによって自動的にインストールされます。ただし、`langchain-core`に依存しないため、必要に応じて単独でインストールして使用することもできます。
LangChainを使用していない場合は、次のコマンドでインストールできます:

```bash
pip install langsmith
```

### ソースから

ソースからパッケージをインストールする場合は、[メインのLangChainリポジトリ](https://github.com/langchain-ai/langchain)をクローンし、インストールしたいパッケージのディレクトリ（`PATH/TO/REPO/langchain/libs/{package}`）を指定して、次のコマンドを実行します。

```bash
pip install -e .
```

LangGraph、LangSmith SDK、および一部の統合パッケージは、LangChainのメインリポジトリとは別に保存されています。[すべてのリポジトリはこちら](https://github.com/langchain-ai)でご覧いただけます。
